# ⚡ Quick Update Guide

## 🎯 3-Minute Update

### 1. Open Your GPT
👉 https://chat.openai.com/gpts → Edit "Website Scanner"

### 2. Update Actions
- **Actions** tab → Delete old action
- **Create new action** → **Import** → Upload `openapi.yaml`
- ✅ Verify new fields appear

### 3. Update Instructions  
- **Create** tab → **Instructions** field
- Open `gpt-instructions.md` → Copy ALL
- Paste into Instructions field
- ✅ Verify it mentions business units, digital goals, job analysis

### 4. Test & Save
- **Preview** panel → Try: "Scan https://example.com"
- Click **Save**

## ✅ What You're Getting

- 60+ CMS detection
- ROI insights (8 categories)
- Business unit detection
- Digital goals analysis
- Job posting analysis
- Role baselines

## 📁 Files Needed

- `openapi.yaml` → Actions tab
