# GPT Conversation Starters

## Recommended Conversation Starters for Custom GPT

These are the suggested prompts that appear when users first interact with your Custom GPT. They should showcase key capabilities and be actionable.

### Option 1: Comprehensive Set (8 starters)

1. **"Scan https://example.com and identify headless CMS opportunities"**
   - Showcases: Website scanning, tech stack detection, ROI analysis

2. **"Analyze these 5 companies: [list URLs] and rank by AI readiness"**
   - Showcases: Batch scanning, stack ranking, AI readiness scoring

3. **"Research [Company Name] - find pricing, security, and tech stack"**
   - Showcases: Research capabilities, page discovery, comprehensive analysis

4. **"What's the digital maturity score for [website]?"**
   - Showcases: Performance analysis, business scale, digital maturity

5. **"Find migration opportunities for [company website]"**
   - Showcases: Legacy system detection, ROI insights, migration analysis

6. **"Analyze LinkedIn profile: [profile URL]"**
   - Showcases: LinkedIn analytics, work patterns, network mapping
   - Note: May get 999 error, but GPT will handle gracefully

7. **"Compare these 3 companies' tech stacks and identify the best headless CMS opportunity"**
   - Showcases: Batch comparison, opportunity scoring, competitive analysis

8. **"What pain points does [company] have that a headless CMS could solve?"**
   - Showcases: Pain point detection, ROI framing, solution mapping

## 📝 How to Add in ChatGPT

1. Open your Custom GPT
2. Go to **Configure** tab
3. Scroll to **Conversation starters** section
4. Click **"Add conversation starter"**
5. Paste each starter (one at a time)
6. Save

## 💡 Tips

- **Keep them concise**: 10-15 words max
- **Be specific**: Include example URLs or placeholders
- **Show variety**: Cover different use cases
- **Action-oriented**: Start with verbs (Scan, Analyze, Research, Compare)
- **Value-focused**: Emphasize outcomes (opportunities, ROI, insights)
