## System Duplication Detection

The scanner identifies when systems have overlapping capabilities:

### Multi-Capability Systems
- **Sanity** can operate as:
  - CMS
  - PIM (Product Information Management)
  - DAM (Digital Asset Management)
  - LMS (Learning Management System)

- **Contentful** can operate as:
  - CMS
  - PIM
  - DAM

### Duplication Scenarios Detected
1. **Multiple CMS systems** - Running multiple CMS platforms simultaneously
2. **Separate PIM + CMS** - Using dedicated PIM alongside CMS (duplicate content management)
3. **Separate DAM + CMS** - Using dedicated DAM alongside CMS (asset management inefficiency)
4. **Multi-capability overlap** - Using Sanity/Contentful for multiple purposes vs. separate systems
